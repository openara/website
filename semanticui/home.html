Construisez de beaux sites web plus rapidement

Semantic est un framework de développement qui permet de créer de beaux sites responsives tout en utilisant un code lisible et simple à comprendre.

Un code HTML simple

Semantic UI traite les mots et les classes comme des concepts interchangeable.

Les Classes utilisent une syntaxe naturelle, comme l'ordre des mots, grâce à une conception intuitive

Obtenez les mêmes bénéfices que BEM ou SMACSS, mais sans l'ennui.

<div class="ui three buttons">
  <button class="ui active button">One</button>
  <button class="ui button">Two</button>
  <button class="ui button">Three</button>
</div>
One
Two
Three

Du Javascript intuitif

Semantic utilise des phrases simples d'appels aux comportements, qui déclencheront les fonctionnalités.

Toute décision arbitraire dans un composant est inclus comme un paramètre que chaque développeur peut modifier.

Run Code
$('select.dropdown')
  .dropdown('set selected', ['meteor', 'ember'])
;
<select name="skills" multiple="" class="ui fluid dropdown">
  <option value="">Skills</option>
  <option value="angular">Angular</option>
  <option value="css">CSS</option>
  <option value="ember">Ember</option>
  <option value="html">HTML</option>
  <option value="javascript">Javascript</option>
  <option value="meteor">Meteor</option>
  <option value="node">NodeJS</option>
</select>

Un débogage simplifié

Grâce à l'enregistrement des performances (log), vous n'avez plus besoin de fouiller des centaines de lignes de log pour trouver une anomalie.

Vous n'avez pas accès à des outils de développement ? Nous vous proposons de visionner cette vidéo.
